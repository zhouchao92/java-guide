#### ArrayList 和 LinkedList 的区别？
|          | ArrayList                                                                      | LinkedList                               |
|----------|--------------------------------------------------------------------------------|------------------------------------------|
| 存储结构 | 数组                                                                           | 链表，存储当前节点的数据和前后节点的引用 |
| 初始容量 | 不指定时，默认为0，在add时分配容量 DEFAULT_CAPACITY = 10；指定大小时按照指定值 |                                          |
| 扩容     | 1.5倍                                                                          |                                          |
| 读       | O(1)                                                                           | O(n)                                     |
| 写       | 尾插法：O(1)，头插法：O(n)                                                     | O(1)                                     |
| 适合场景 | 读多写少                                                                       | 写多读少                                 |
